Data Types

A data type is an attribute of data which is defined by the values it can take. It also allows the compiler/interpreter to understand how the programmer wants to use the data.

String
- stores characters surounded by double quotes, "hello"

int
- stores whole numbers such as 1, 257, 1050

float
- stores floting point numbers such as 1.5, 2.57, 105.015

char
- stores individual characters in single quotes such as 'a', 'f', 'x'

boolean
- stores values which represent two states: true or false
