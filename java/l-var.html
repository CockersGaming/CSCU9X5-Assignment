Data Types

A data type is an attribute of data which is defined by the values it can take. It also allows the compiler/interpreter to understand how the programmer wants to use the data.

String
- stores characters surounded by double quotes, "hello"

int
- stores whole numbers such as 1, 257, 1050

float
- stores floting point numbers such as 1.5, 2.57, 105.015

char
- stores individual characters in single quotes such as 'a', 'f', 'x'

boolean
- stores values which represent two states: true or false

Declaring a Variable

Bellow shows how to declare variables in Java. We will use the data types you learnt on the previous page.

Remember to end a line of code in Java with a semi-colon!


Declaring a String
String exampleName = "Hello World";
Declaring an Integer
int exampleName = 123;
Declaring a Float
float exampleName = 123.456;
Declaring a char
char exampleName = 'h';
Declaring a Boolean
boolean exampleName = false;

As you can see we state the data type first then type the name of the variable. We can then assign that variable a value.

How to use variables to perform maths

Variables are used to store data. However we can use variables to perform actions such as addition, division, splitting strings, etc...

First we need to declare two variables
int exampleNumber1 = 6;
int exampleNumber2 = 8;
To add these we can create a new variable called sum and then add the two variables together using the + oporator
int sum = exampleNumber1 + exampleNumber2;
If we were to print sum we would get 16
